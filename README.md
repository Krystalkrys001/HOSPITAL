# KRYSTAL KRONA CARE — Hospital Website

A fully responsive hospital and healthcare website built with pure HTML, CSS and JavaScript.



## ✨ Features
- Clean professional medical design
- Services section with 5 specialties — Cardiology, Pediatrics, General Medicine, Orthopedics, Dermatology
- Interactive service modals with detailed department info
- Appointment booking form
- Patient testimonials section
- Admin login & dashboard system
- Mobile responsive with hamburger menu
- Smooth scroll navigation

## 🏥 Medical Services Covered
- ❤️ Cardiology — Heart & cardiovascular disease treatment
- 👶 Pediatrics — Healthcare for infants, children & adolescents
- 🏥 General Medicine — Primary care & common conditions
- 🦴 Orthopedics — Bone, joint & musculoskeletal disorders
- 💆 Dermatology — Skin conditions & cosmetic dermatology

## 🛠 Built With
- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Poppins & Playfair Display)
